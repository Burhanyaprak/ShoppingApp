## Features

- Splash
- Onboarding
- Authentication
- Product list
- Basket
- Searching
- Profile

## Tech Stack

**Client:** Kotlin, Jetpack Navigation, ViewModel, Flow, Retrofit, Dagger-Hilt

**Server:** Google Firebase

**Local Database Library:** Room Database

**Version Control System:** Git
